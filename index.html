Description

A guided walk-through of the simulator

Start here

This simulator is designed to help you move through a full credit risk workflow in a clear order, from portfolio intake through rating model design, validation, monitoring, calibration, and Basel capital views.

What this simulator does

Turn a complex workflow into a step-by-step experience

  • Load a portfolio or start with the bundled demo dataset.
  • Map the required fields once and reuse them across all downstream tabs.
  • Prepare features, estimate a scorecard, test performance, and inspect capital outputs.
  • Export synchronized R code and methodology as you move through the workflow.

Recommended path

Follow the tabs from left to right

  1. Begin with Description to understand the logic and expected outputs.
  2. Open Portfolio Workspace to upload data, map fields, and inspect the portfolio.
  3. Continue through the modeling tabs in sequence so each result builds on the previous one.
  4. Use the monitoring, calibration, and Basel views once the scorecard is ready.

Best first run

Get familiar in a few minutes

If this is your first visit, load the demo portfolio, confirm the suggested mappings, and walk through each tab once without changing too many parameters. That gives you a fast end-to-end view before you start testing your own portfolio assumptions.

What you will get

Operational outputs, not just visuals

  • Interactive diagnostics for feature quality, model performance, and portfolio drift.
  • Portable R code for each stage of the workflow.
  • Methodology text that stays aligned with the active simulator state.
  • Basel III and Basel IV views that make capital implications easier to explain.
